### AD Basics Notes

* **LDAP**: Lightweight Directory Access Protocol, the database representing AD trees.
* **Kerberos**: Ticket-based network authentication protocol.
* **GPO**: Group Policy Objects used by administrators to configure workstations domain-wide.
